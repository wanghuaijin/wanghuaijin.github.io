1. Introduction to minimax approximation

    The best minimax approximation from a set A\mathscr{A}A to a function fff in C[a,b]C[a,b]C[a,b] is the element of A\mathscr{A}A that minimizes the expression

∥f−p∥∞=max⁡a⩽x⩽b∣f(x)−p(x)∣,p∈A.(1.1) \|f-p\|_{\infty}=\max _{a \leqslant x \leqslant b}|f(x)-p(x)|, \quad p \in \mathscr{A}. \hspace{3em} \tag{1.1} ∥f−p∥∞​=a⩽x⩽bmax​∣f(x)−p(x)∣,p∈A.(1.1)

In this text, we study the conditions that are satisfied by a best approximation, when A\mathscr{A}A is a linear space.

    If A\mathscr{A}A is the space Pn\mathbb{P}_nPn​ of algebraic polynomials of degree at most nnn, it reduces to a particularly simple form. In this text, we consider a more general case when A\mathscr{A}A satisfies the 'Haar condition'. In this case, we consider:

  1. properties of best minimax approximation, including that the best approximation is unique;
  2. an excellent method for calculating best approximations, which is called the exchange algorithm.

Start with a question

    The theory that is developed for the case when A\mathscr{A}A is any finite dimensional linear comes from asking the following question:

let p∗p^*p∗ be a trial approximation from A\mathscr{A}A to fff. Can we find a change to p∗p^*p∗ that reduces the maxmimum error of the trial approximation?

    In other words, we seek an element p∈Ap\in\mathscr{A}p∈A such that the inequality

∥f−(p∗+θp)∥∞<∥f−p∗∥∞(1.2) \left\|f-\left(p^*+\theta p\right)\right\|_{\infty}<\left\|f-p^*\right\|_{\infty} \hspace{3em} \tag{1.2} ∥f−(p∗+θp)∥∞​<∥f−p∗∥∞​(1.2)

is satisfied for some value of the scalar parameter θ\thetaθ.

3. The characterization theorem and the Haar condition

   If the set A\mathscr{A}A of approximating functions is the space Pn\mathbb{P}_nPn​ of algebraic polynomials of degree at most nnn, then it is rather easy to test wether condition (1.6) can be obtained. We make use of the fact that a function in Pn\mathbb{P}_nPn​ has at most nnn sign changes. Therefore, if the error function [f(x)−p∗(x)][f(x) - p^*(x)][f(x)−p∗(x)] changes sign more than nnn times as xxx ranges over ZM\mathscr{Z}_MZM​, then p∗p^*p∗ is a best approximation. Conversely, if the number of sign changes does not exceed nnn, then we can choose the zeros of a polynomial in Pn\mathbb{P}_nPn​ so that condition (1.6) is satisfied. This result is usually called the minimax characterization theorem, and it is stated formally below.

Haar condition

   It is useful to express the theorem in a form that applies to a class of functions that includes polynomials as a special case. The usual way of defining this class is to identify the properties of polynomials that are used in the proof of the characterization theorem. They are the following two conditions:

  1. If an element of Pn\mathbb{P}_nPn​ has more than nnn zeros, then it is identically zero.
  2. Let {ζj;j=1,2,…,k}\left\{\zeta_j ; j=1,2, \ldots, k\right\}{ζj​;j=1,2,…,k} be any set of distinct points in the open interval (a,b)(a,b)(a,b), where k⩽nk\leqslant nk⩽n. There exists an element of Pn\mathbb{P}_nPn​ that changes sign at these points, and that has no other zeros. Moreover, there is a function in Pn\mathbb{P}_nPn​ that has no zeros in [a,b][a,b][a,b].

Condition 2 states that there exists two kind of polynomials in Pn\mathbb{P}_nPn​ such that only kkk roots at which sign changes contained in (a,b)(a,b)(a,b), or no zeros in [a,b][a,b][a,b].

  1. If a function in Pn\mathbb{P}_nPn​ that is not identically zero, has jjj zeros, and if kkk of these zeros are interior points of [a,b][a,b][a,b] at which the function does not change sign, then the number (j+k)(j+k)(j+k) is not greater than nnn.

  2. Let {ξj:j=0,1,…,n}\left\{\xi_j : j=0,1, \ldots, n\right\}{ξj​:j=0,1,…,n} be any set of distinct points in [a,b][a,b][a,b], and let {ϕi:i=0,1,…,n}\left\{\phi_i : i=0,1, \ldots, n\right\}{ϕi​:i=0,1,…,n} be any basis of Pn\mathbb{P}_nPn​. Then the (n+1)×(n+1)(n+1)\times(n+1)(n+1)×(n+1) matrix whose elements have the values {ϕi(ξj):0⩽i,j⩽n}\{\phi_i (\xi_j): 0\leqslant i,j \leqslant n\}{ϕi​(ξj​):0⩽i,j⩽n} is non-singular.

   An (n+1)−(n+1)-(n+1)−dimensional linear subspace A\mathscr{A}A of C[a,b]C[a,b]C[a,b] is said to satisfy the 'Haar condition' if these four statements remain true when P\mathbb{P}P is replaced by the set A\mathscr{A}A.

   Equivalently, any basis of A\mathscr{A}A is called a 'Chebyshev set'.

Spaces that satisfy the Haar conditiion are studied in Appendix A. It is proved that properties 1, 3 and 4 are equivalent, and that these properties imply condition 2.

It is usual to define the Haar condition in terms of the first property.

Thus, A\mathscr{A}A satisfies the Haar condition if and only if, for every non-zero ppp in A\mathscr{A}A, the number of roots of the equation {p(x)=0:a⩽x⩽b}\{p(x) =0: a \leqslant x \leqslant b\}{p(x)=0:a⩽x⩽b} is less than the dimension of A\mathscr{A}A.
